# petclinic-ui-react

## install

prerequisite: npm login

```bash
$ npm login --scope=@t-kiyono --registry=https://npm.pkg.github.com
> Username: USERNAME
> Password: TOKEN
> Email: PUBLIC-EMAIL-ADDRESS
```

run the following command

```bash
$ npm install
```

## build

```bash
$ npm run build
```

## development

```bash
$ npm start
```
